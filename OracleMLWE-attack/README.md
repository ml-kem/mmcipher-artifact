# OracleMLWE-mmKEM-Cryptanalysis

This repository contains a **SageMath** implementation that reproduces our cryptanalytic attack on the mmKEM construction from the Asiacrypt 2025 paper:  
[*Lattice-based Multi-message Multi-recipient KEM/PKE with Malicious Security*](https://eprint.iacr.org/2025/1655).

## Run
```bash
sage attack_demo.sage 
```
